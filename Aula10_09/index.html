Elementos e marcação semântica em HTML

O que é semântica?

O que é semântica? A semântica é a área da linguística que estuda o significado e a sua relação com o significante.

O que é semântica HTML?

Semântica é o estudo do significado e interpretação de palavras. Aplicando isso ao HTML, significa a melhor compreensão do HTML para o computador. Antes do HTML5, utilizava-se somente divs para dividir conteúdos e classes para estilizar, sem contar com a compreensão do próprio computador para “ entender ” o HTML.

O que são elementos semânticos HTML?

elementos semânticos são elementos com um significado. HTML semântica é o uso de marcação HTML para reforçar a semântica, ou significado, da informação em páginas web e aplicações web em vez de meramente para definir a sua apresentação ou olhar.

não semânticos: div> e span - Não diz nada sobre o seu conteúdo.

semânticos: form , table e img - define claramente o seu conteúdo.

Cite alguns dos benefícios de se utilizar elementos semânticos (marcação semântica).

Usando html semântico para marcar suas páginas, você pode criar páginas que são mais acessíveis, tanto para as pessoas com deficiência, bem como aos motores de busca. Marcação semântica ajuda os motores de busca determinar qual é a relevância do conteúdo de uma página.
